# Scrap Mechainc Logic Browser
Fanmade website that generates Scrap Mechanic blueprints.

### Purpose
Simplify creating massive logic projects & reduce repeatition.

### Controls
- Left Click - Wires nodes.
- Right Click - Moves nodes/camera
- E - Interact
- F - Cycle Mode
- X - Delete
- B - Export to clipboard

### Documentation
- Layers are used as blueprints having a node of their own to reduce repeative actions, the node will execute the layers logic gates to get an output.
- Nodes are just draggable connectable, they serve as an object with the very basic functionality.
